# web_vpp
# WEB_VPP – Website bán văn phòng phẩm (PHP MVC)

## Giới thiệu dự án

**WEB_VPP** là một dự án website thương mại điện tử bán **văn phòng phẩm**, được phát triển với mục tiêu thể hiện khả năng **lập trình PHP nâng cao** kết hợp cùng **HTML, CSS và Bootstrap**, áp dụng mô hình **MVC (Model – View – Controller)** để xây dựng hệ thống rõ ràng, dễ mở rộng và bảo trì.

Dự án được thực hiện nhằm phục vụ cho **học tập, báo cáo môn học**, đồng thời được sử dụng như **sản phẩm minh chứng năng lực cá nhân** trong hồ sơ xin việc (CV / Portfolio).

---

## Công nghệ sử dụng

- **Ngôn ngữ Backend:** PHP (hướng đối tượng, lập trình nâng cao)
- **Mô hình kiến trúc:** MVC (Model – View – Controller)
- **Frontend:** HTML5, CSS3
- **Framework giao diện:** Bootstrap
- **Cơ sở dữ liệu:** MySQL
- **Thư viện hỗ trợ:** Composer, PHPMailer
- **Quản lý mã nguồn:** Git & GitHub

---

## Kiến trúc hệ thống

Dự án được tổ chức theo mô hình **MVC**, trong đó:

- **Model:** Xử lý nghiệp vụ, truy vấn cơ sở dữ liệu, tách biệt hoàn toàn với giao diện
- **View:** Giao diện người dùng (Frontend & Backend), sử dụng HTML, CSS và Bootstrap
- **Controller:** Điều hướng luồng xử lý, nhận request từ người dùng và gọi Model tương ứng

Cách tổ chức này giúp:
- Mã nguồn rõ ràng, dễ đọc
- Dễ mở rộng tính năng
- Dễ bảo trì và nâng cấp trong tương lai

- Quản lý đơn hàng, chi tiết đơn hàng
- Quản lý người dùng
- Quản lý banner, tin tức, trang nội dung

- Hệ thống chat hỗ trợ người dùng
